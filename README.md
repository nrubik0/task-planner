# task-planner
Простой таск‑планнер на React Native (Expo).

## Быстрый старт

1. Установите зависимости
```bash
npm install
```

2. Запустите приложение в режиме разработки

```bash
npx expo start
```

3. Откройте приложение
- Android emulator (Android Studio)
- iOS simulator (Xcode)
- Expo Go — сканируйте QR-код в терминале/веб-интерфейсе Expo

## Структура проекта

- app/ — точка входа (Expo)
- src/components — UI‑компоненты (CalendarGrid, DraggableTask и т.д.)
- src/screens — экраны (QuickAddScreen, CalendarScreen)
- src/store — Zustand store (персистентность через AsyncStorage)
- src/utils — парсер задач (taskParser.ts), suggestions.ts

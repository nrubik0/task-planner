# task-planner
Простой таск‑планнер на React Native (Expo).

## Быстрый старт

1. Установите зависимости
```bash
npm install
```

2. Запустите приложение в режиме разработки

```bash
npx expo start
```

3. Откройте приложение
- Android emulator (Android Studio)
- iOS simulator (Xcode)
- Expo Go — сканируйте QR-код в терминале/веб-интерфейсе Expo

## Структура проекта

- app/ — точка входа (Expo)
- src/components — UI‑компоненты (CalendarGrid, DraggableTask и т.д.)
- src/screens — экраны (QuickAddScreen, CalendarScreen)
- src/store — Zustand store (персистентность через AsyncStorage)
- src/utils — парсер задач (taskParser.ts), suggestions.ts

# task-planner

Simple task planner built with React Native (Expo).

## Quick start

1. Install dependencies

```bash
npm install
```

2. Start the app in development

```bash
npx expo start
```

3. Open the app
- Android emulator (Android Studio)
- iOS simulator (Xcode)
- Expo Go — scan the QR code from the terminal / Expo web UI

## Project structure

- app/ — entry point (Expo)
- src/components — UI components (CalendarGrid, DraggableTask, etc.)
- src/screens — screens (QuickAddScreen, CalendarScreen)
- src/store — Zustand store (persisted via AsyncStorage)
- src/utils — task parser (taskParser.ts), suggestions.ts
